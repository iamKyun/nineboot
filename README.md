# nineboot

This app is a tool for *ninebot* bikes, it can change your bike's running mode.

## How To Use
1. Download the latest release and install it.
2. Open the app, and then search your device.
3. Select your device, and then press "SEND" button.

After you pressed the "SEND" button, you may hear a "bee" sound from your bike, 
it means you successfully change your bike's running mode.
